Comboios de Portugal

The Comboios de Portugal website redesign was developed for the Web Accessibility and Usability course, with the aim of improving accessibility, usability, and overall user experience. I chose this project due to the site's significant usability issues. Through maze testing and user interviews, I identified key pain points and gathered insights that informed the UX improvements. I then created high-fidelity prototypes focused on delivering a more intuitive, accessible, and user-centered interface.
